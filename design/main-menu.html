01 Color

Borders, numbers and paths drop to neutral gray. Orange chrome everywhere → accent reserved for the one selected row. Hierarchy for free.

02 Selection

A full-width wash + a solid left rail replace the half-painted bar, so the cursor never looks like a render glitch.

03 Top controls

The two stacked chevron rows get tiny AGENT / PLAN labels and only light up the one that's focused — no more cramped collision with the title.

04 Tabs

The ▌Projects▐ glyph artifact becomes a clean underline on the active tab — reads as a tab, not a bug.

05 Worktrees

The count becomes a small pill pinned beside its own row, so it can't be misread as belonging to the row above.
